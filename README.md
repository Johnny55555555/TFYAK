# Вариант №58
Объявление массива символов с инициализацией строковой константой на языке С


## Синтаксическая конструкция

Объявление символьного массива:

```
const char идентификатор[размер] = "строка";
```

## Примеры корректных строк

```
const char text[5] = "Hello";
const char name[10] = "Test";
const char message[12] = "Hello world";
```

## Допустимые лексемы

| Лексема       | Описание              |
| ------------- | --------------------- |
| const         | ключевое слово        |
| char          | тип данных            |
| идентификатор | имя переменной        |
| число         | размер массива        |
| строка        | значение              |
| [ ]           | скобки                |
| =             | оператор присваивания |
| ;             | конец оператора       |

# Разработка грамматики

G[z] = (Vt, Vn, Z, P)

### Терминальные символы

```
Vt = {const, char, [, ], =, letter, digit, space, symbols, ;, _}
```

### Нетерминальные символы

```
Vn = {<start>, <space>, <type>, <spaceafter>, <ID>, <IDREM>, <COUNT>, <COUNTREM>, <ASSIGN>, <string>, <strsym>, <strrem>, <stop>}
```

### Начальный символ

```
Z = <start>
```

### Правила грамматики

```
1) <start> -> "const" <space>
2) <space> -> "_" <type>
3) <type> -> "char" <spaceafter>
4) <spaceafter> -> "_" <ID>
5) <ID> -> "letter" <IDREM>
6) <IDREM> -> "letter" <IDREM> | "digit" <IDREM> | "[" <COUNT>
7) <COUNT> -> "digit" <COUNT> | "]" <COUNTREM>
8) <COUNTREM> -> "=" <ASSIGN>
9) <ASSIGN> -> """ <string>
10) <string> -> "letter" <strsym>
11) <strsym> -> "symbols" <strsym> | """ <strrem>
12) <strrem> -> ";" <stop>
```

# Метод анализа

Для реализации синтаксического анализатора используется:

**Метод Айронса (рекурсивный спуск)**

Алгоритм анализа:

```
start
 ↓
const
 ↓
char
 ↓
identifier
 ↓
[
 ↓
number
 ↓
]
 ↓
=
 ↓
string
 ↓
;
```

Каждый нетерминал реализован отдельным методом синтаксического анализатора.

# Диагностика и нейтрализация синтаксических ошибок

В программе реализовано:

* обнаружение синтаксических ошибок
* продолжение анализа после ошибки
* метод Айронса
* нейтрализация ошибок

Типы обнаруживаемых ошибок:

* отсутствие ключевого слова const
* отсутствие char
* ошибка идентификатора
* ошибка размера массива
* отсутствие оператора =
* отсутствие строки
* отсутствие ;

При обнаружении ошибки:

* ошибка заносится в таблицу
* анализ продолжается со следующей строки

## Строка с несколькими ошибками

```
const char name[] = "Test";
char name[5] = "Test";
```

Результат:

| Фрагмент | Строка | Описание        |
| -------- | ------ | --------------- |
| ]        | 1      | Ожидалось число |
| char     | 2      | Ожидался const  |
